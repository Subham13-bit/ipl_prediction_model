# IPL Winner Prediction

Machine Learning project to predict IPL match outcomes using historical match data.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Model Performance
- Train Score: 0.99
- Test Score: 0.95
- R2 Score: 0.95

## Features Used
- Team1
- Team2
- Toss Winner
- Venue
- Target Score
